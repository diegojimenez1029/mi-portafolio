Sobre Mí

Soy Diego Jiménez, desarrollador con experiencia en el desarrollo de aplicaciones web para el sector salud, especialmente en Hospital Luis Tisné. He trabajado creando soluciones internas como sistemas de intranet, módulos para estadísticas con dashboards Power BI, y plataformas de tickets para departamentos como Calidad.

🔧 Mis herramientas principales incluyen Python (Django), JavaScript, PHP, HTML, CSS y Bootstrap. Tengo experiencia sólida con bases de datos como Informix, MySQL y PostgreSQL.

🧪 También he participado activamente en proyectos de interoperabilidad usando Mirth Connect, desarrollando canales para integración de sistemas mediante HL7, APIs REST, SOAP y protocolos TCP/IP.

Actualmente estoy ampliando mis conocimientos en Node.js y preparando mi experiencia para nuevos desafíos internacionales.
